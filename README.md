# Rock Paper Scissors Web Project

A small interactive web project built with **HTML, CSS, and JavaScript**.
The project includes multiple features designed to demonstrate core front-end development skills such as DOM manipulation, event handling, and responsive UI design.

## Features

* 🎮 **Rock Paper Scissors Game** – Play against the computer with score tracking for wins, losses, and ties.
* ⏱ **Stopwatch** – Start, pause, and reset a stopwatch in real time.
* 🖥 **Live Digital Clock** – Displays the current time dynamically.
* 🎨 Clean and simple user interface built with CSS.

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run the Project

1. Clone the repository
2. Open the `index.html` file in your browser
3. Start playing the game or use the stopwatch

## Purpose of the Project

This project was built as part of my front-end development learning journey and demonstrates practical JavaScript functionality and UI interaction.
